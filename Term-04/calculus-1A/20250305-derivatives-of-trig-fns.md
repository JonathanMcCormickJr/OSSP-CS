# Derivatives of Trigonometric Functions

| Trig     | Trig'         |
|----------|---------------|
| sin(x)   | cos(x)        |
| tan(x)   | sec^2 (x)     |
| sec(x)   | sec(x)tan(x)  |
| cos(x)   | -sin(x)       |
| cot(x)   | -csc^2 (x)    |
| csc(x)   | -csc(x)cot(x) |

## Pythagorian Trig Identity
`sin^2 (x) + cox^2 (x) = 1`
